# vim常用命令

vim filename	打开或者新建一个文件

?xx 向下查询匹配字符串xx

?N 查找下一个

?n 查找上一个

i 进入编辑模式

: 进入命令模式

:w 保存

:q 退出

:wq 保存退出

# Linux常用命令

修改文件权限	chomp 777 filename 

查看进程	ps aux

查看redis相关进程	ps aux | grep redis
